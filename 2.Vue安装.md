```vue
CND方式,不能用于大型项目和生产环境
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
```



安装

```shell
安装16.0 or height 版本的Node.js
打开命令行执行

npm init vue@latest
这条指令会创建vue，属于官方脚手架工具,会有一些可选功能,可选功能以后能随时添加

cd  项目路径
切换到项目路径

npm install 
安装必须要的内容

npm run dev
启动项目

如果部署生产环境时请执行
npm run build
```



使用Vite创建

```
创建工程
npm  init vite-app 项目名

进入项目
cd 项目名

npm install 
安装依赖

npm run dev
启动项目

```



项目打包

```shell
cnpm run build
```







随便写一个东西

```html
<!--author:steam-404-->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>example02</title>
    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
</head>
<body>
<div id="message">
    <!--    {{vue的语法:变量}}-->
    <p>{{age}}</p>
    <p>{{name}}</p>
</div>
<script>
    const ikun={//配置对象
        data:function (){
            return{
                age:23,
                name:"蔡徐坤"
            }
        }
    }
    Vue.createApp(ikun).mount("#message")//创建应用,将配置对象传入
</script>
</body>
</html>
```
